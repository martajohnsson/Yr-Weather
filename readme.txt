== Description ==


== Installation ==

1. Put files in WP_CONTENT_DIR . '/plugins/yr-weather'
1. Create the folder WP_CONTENT_DIR . '/cache/yr_cache'
1. Make yr_cache writable by the web server
1. Activate the plugin
1. In a post or page insert: [wpyr url="http://www.yr.no/place/Denmark/North_Jutland/Skagen/" name="Skagen" flagdays="Tue Thu Sat"]
